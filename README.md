# 🛍️ Retail Sales & Customer Behavior Analysis Dashboard

An end-to-end **Data Analytics & Business Intelligence project** focused on analyzing retail sales performance, customer purchasing behavior, and profitability trends using **Python, SQL, and Power BI**.

---

# 📌 Project Overview

This project transforms raw retail sales data into meaningful business insights through:

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- SQL-Based Business Analysis
- Customer Behavior Analysis
- Interactive Power BI Dashboarding

The goal of this project is to help businesses understand:

- Sales performance trends
- Regional profitability
- Customer purchasing behavior
- Product category contribution
- Profit margin performance
